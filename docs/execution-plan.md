# 分阶段执行计划 — 番茄钟专注学习软件

## 执行原则

- 每阶段一次只做一件事，完成并验证后再进入下一阶段
- 每阶段结束后记录开发日志
- 不跳步、不并行做多阶段工作
- 每个阶段都可独立运行和测试

---

## 阶段 0：项目初始化

- [ ] 创建 Xcode 项目 `PomodoroTimer`（macOS App, SwiftUI）
- [ ] 配置 Info.plist: `LSUIElement = YES`
- [ ] 创建项目目录结构（Models / Services / Views / Components）
- [ ] 添加 SF Symbol 图标资源

**验证**：项目 Build 成功，运行后菜单栏出现图标

---
