# 项目需求文档 — 番茄钟专注学习软件

## 1. 项目概述

一款运行在 macOS 上的番茄钟专注学习软件，帮助用户记录和管理每次专注学习的时长。

## 2. 用户画像

本科生，略懂代码，需要一款简洁直观的专注计时工具。

## 3. 功能需求

### 3.1 计时模式

| 需求 | 说明 |
|------|------|
| 倒计时模式 | 默认 25 分钟，支持自定义时长（预设 + 手动输入） |
| 正计时模式 | 从 00:00 开始向上累计，手动点击结束 |

### 3.2 专注记录

- 每次从"开始"到"结束"自动保存一条记录
- 记录包含：分类标签、模式、开始时间、实际时长、目标时长
- 数据保存至 `/Users/why/Desktop/番茄钟/Data/sessions.json`

### 3.3 历史记录

- 按时间倒序展示所有记录
- 支持按分类筛选
- 支持按日期范围搜索
- 支持删除单条记录

### 3.4 数据统计

| 维度 | 展示方式 |
|------|---------|
| 单日 | 当日各分类时长汇总 |
| 一周 | 近 7 天每日总时长条形图 |
| 一月 | 近 4 周每周总时长条形图 |
| 一年 | 近 12 月每月总时长条形图 |

### 3.5 专注分类

- 预设分类：学习、阅读、编程、背单词、其他
- 支持添加/编辑/删除自定义分类
- 每条记录必须选择一个分类

### 3.6 通知提醒

- 倒计时归零时发送系统通知

## 4. 非功能需求

| 项目 | 要求 |
|------|------|
| 平台 | macOS 14+ |
| 界面形式 | 菜单栏常驻（状态栏图标 + Popover 面板） |
| UI 风格 | 简洁直观，淡蓝色主色调 |
| 数据存储 | 本地 JSON 文件 |

## 5. 暂不包含的需求（后续迭代）

- 独立应用窗口
- 多设备同步
- 番茄钟周期（专注→休息→专注）
- 白噪音
- 导出数据
- iCloud 同步
