# UI 设计规范 — 番茄钟专注学习软件

## 1. 设计原则

- **简洁直观**：信息层级清晰，操作路径最短
- **减少干扰**：不必要的信息不展示
- **反馈明确**：每次操作有即时视觉反馈

## 2. 色彩系统

| 用途 | 颜色 | 值 |
|------|------|-----|
| 主色调 | 淡蓝 | `Color.blue.opacity(0.15)` / `#D6EAF8` |
| 强调色 | 标准蓝 | `Color.blue` |
| 文字主色 | 深灰 | `.primary` |
| 文字辅色 | 中灰 | `.secondary` |
| 背景色 | 浅灰白 | `.windowBackgroundColor` |
| 危险操作 | 红色 | `.red`（仅删除按钮） |

## 3. 排版

| 元素 | 字体 | 大小 |
|------|------|------|
| 计时数字 | 等宽数字 | `.system(size: 48, weight: .light, design: .monospaced)` |
| 页面标题 | 系统默认 | `.title2` |
| 正文/标签 | 系统默认 | `.body` / `.caption` |
| 按钮文字 | 系统默认 | `.headline` |

## 4. 布局规范

| 区域 | 尺寸/规则 |
|------|----------|
| Popover 整体 | 宽 360pt，高 480pt |
| Tab 栏 | 顶部或底部，标准高度 |
| 计时圆形环 | 直径 200pt |
| 按钮 | 最小点击区域 44×44pt |
| 卡片圆角 | 12pt |
| 列表项高度 | 约 60pt |
| 条形图高度 | 200pt |

## 5. 组件状态

### 开始/结束按钮
- 未开始：蓝色背景 "开始专注"
- 计时中：红色背景 "结束专注"
- 禁用态：未选分类时按钮置灰

### 计时显示
- 倒计时：`MM:SS` 格式，圆形进度环显示剩余比例
- 正计时：`MM:SS` 格式，无进度环，纯数字递增

### 预设时长按钮
- 默认选中 25 分钟
- 选中态：蓝色填充
- 未选中：蓝色边框

## 6. 动效

- 计时数字切换：无动画（避免视觉干扰）
- 进度环：线性过渡，跟随秒数变化
- 页面切换：系统默认 TabView 滑动
- 弹出/收起：Popover 默认动画
